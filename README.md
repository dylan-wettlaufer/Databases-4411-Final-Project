Android Notes App with Database Performance Testing
A fully-featured notes application for Android that demonstrates database performance optimization with SQLite indexing and query planning.

Quick Start Guide
Prerequisites
Android Studio (latest version)

Android SDK (API 21+)

Java 8+

Android device/emulator

Installation
Open in Android Studio

Sync Gradle (automatic)

Run on device/emulator (Shift + F10)

Main Features
Create, Read, Update, Delete notes

Search with performance comparison (slow vs fast)

Bulk data generation (50k notes)

Query plan analysis

Real-time search

How to Use
Basic Operations
Add Note: Tap + button

Edit Note: Tap any note

Delete Note: Long-press note

Search: Type in search bar

Performance Testing
Tap "Generate 50k Notes" to populate database

Tap "Run Benchmark" to compare search speeds

Results appear in Toast and Logcat

Use "Clear All" to reset database
